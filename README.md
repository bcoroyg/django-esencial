# DJANGO ESENCIAL

## Entorno Virtual - Python 3
* Crear
```
python3 -m venv venv
```

* Iniciar
```
source venv/bin/activate
```

* Desactivar
```
deactivate
```

* Lista de dependencias del entorno
```
pip freeze
```
